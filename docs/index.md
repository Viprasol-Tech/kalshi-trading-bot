# Kalshi Trading Bot

Open-source algorithmic trading framework for [Kalshi](https://kalshi.com)
prediction markets — built and maintained by [Viprasol Tech](https://viprasol.com).

!!! warning "Disclaimer"
    This software is for **educational purposes only** and is **not financial
    advice**. Trading event contracts involves substantial risk, including the
    total loss of capital. Always start in **dry-run / demo** mode. Viprasol Tech
    is not affiliated with or endorsed by Kalshi.

## What you get

- RSA-PSS authenticated REST client and WebSocket streaming.
- A pluggable strategy interface with example strategies.
- A risk manager (pre-trade limits + fractional Kelly sizing).
- A backtester for replaying market snapshots.
- A Typer CLI that runs strategies in dry-run by default.
